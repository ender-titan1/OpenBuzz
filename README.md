# OpenBuzz
OpenBuzz is a free and open source buzzer system for trivia games & tournaments, gameshows, or any competition where timing matters. It can be used with 2 teams of up to 4 players or with up to 8 independent players.

## Features
- Open source 
- Fully hand-solderable
- Modular and modifyable
- Designed to be fabricated as cheaply as possible
- Support for extensive customization, many game modes and a wide range of buzzer sounds
- Use of common RJ45 connector for easy and cheap connection between base and pods

## Why make this?
Professional buzzer systems tend to be very expensive and not very customizable, as well as being tricky to obtain due to overseas shipping. OpenBuzz aims to provide a cheaper and more flexible alternative, while maintaining fairness and accurate winner detection. It was originally designed for a local trivia team that needed a system to practice with, but since has been made open source. As a bonus it allows players to have their own custom buzzers, ranging from simple buttons to way over-the-top triggers.

# Usage
1. Download schematic and PCB files
2. Modify freely to fit your needs, components, etc...
3. Fabricate with any online service for as little as <$10
4. Assemble base and as many pods as you need
5. 3D-print enclosures if needed
6. Use in any game you need it for!

**Disclaimer: V1 is included in this repository for historical reference only. Versions V2 and above are the ones intended for use.**

PCB files and schematics are provided as KiCad files. 3D files for enclosures are provided as STEP files.
